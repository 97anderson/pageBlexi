# BLEXI Landing Page

Landing page de BLEXI (Next.js + Tailwind) enfocada en conversión hacia WhatsApp.

## Stack

- Next.js (App Router)
- React + TypeScript
- TailwindCSS
- Framer Motion

## Requisitos

- Node.js 18+ (recomendado 20+)
- npm

## Desarrollo local

```bash
npm install
npm run dev
```

Abrir `http://localhost:3000`.

## Scripts

```bash
npm run lint
npm run build
npm run start
```
